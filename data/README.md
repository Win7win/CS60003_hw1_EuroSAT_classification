# Dataset

Place the **EuroSAT RGB** dataset here so the directory looks like:

```
data/
└── EuroSAT_RGB/
    ├── AnnualCrop/
    ├── Forest/
    ├── HerbaceousVegetation/
    ├── Highway/
    ├── Industrial/
    ├── Pasture/
    ├── PermanentCrop/
    ├── Residential/
    ├── River/
    └── SeaLake/
```

The dataset is not included in this repository due to size.  
Download from: https://github.com/phelber/EuroSAT
